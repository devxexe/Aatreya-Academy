Academic Roadmap

The progressive path of learning from Nursery to Senior Secondary.

Foundation Stage

Nursery - Class II

  • Activity-Based Learning
  • Motor & Cognitive Skills
  • Phonetic foundations
Preparatory Stage

Class III - V

  • Mathematical Logic
  • Conceptual Literacy
  • Environmental Studies
Middle Stage

Class VI - VIII

  • Scientific Lab Inquiry
  • Coding & Digital Skills
  • Project-Based Learning
Secondary Stage

Class IX - X

  • Academic Specialization
  • Board Exam Readiness
  • Analytical Research
Senior Secondary

Class XI - XII

  • Sci / Comm / Arts Streams
  • Competitive Exam Prep
  • Leadership Development
